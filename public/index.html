Inv–noice API


A small serverless Node.js API. Hosted with Vercel.
See the docs for more info.


A brief overview of our api routes:

  • `POST /api/v1/invoices` - Create an invoice.
  • `GET /api/v1/invoices` - List all invoices.
  • `GET /api/v1/invoices/:invoice_id` - View invoice details.
  • `PATCH /api/v1/invoices/:invoice_id` - Update an invoice.
  • `DELETE /api/v1/invoices/:invoice_id` - Delete an invoice.
